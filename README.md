### Tech
* Client React/Redux based on this boilerplate code https://github.com/coryhouse/react-slingshot

### Installation

#### Client
* Install the latest node https://nodejs.org/
*    ```sh
        $ git clone https://github.com/DVLP/react-slingshot.git

        $ cd react-slingshot

        $ npm install

        $ npm start
        ```
### Testing
*    ```sh
        $ npm test
        ```
